1. Change Directory: cd directoryName
2. Create a Directory: mkdir directoryName
3. Rename a Directory: mv oldDirectoryName newDirectoryName
4. List the contents of a directory: ls directoryName
5. Move a file to a new location: mv oldPath/file newPath
6. Copy a file to a new location: cp oldPath/file newPath
7. Search for a string in a file: grep string file.text
8. Access the shell of a remote machine: ssh
9. Remove a directory folder: rm -r directoryName
